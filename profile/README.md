# Fire Tools

Fire Tools builds privacy-first personal finance tools that keep your data under your control.

Our main app is available in the [Fire Tools app repository](https://github.com/fire-tools-inc/app).

## Mobile apps coming soon

iOS and Android apps are coming soon. At launch, they will work without a central Fire Tools server and will communicate only with the app running on your own network.

If you want to connect from outside your home network, you will need to configure and provide a secure tunnel to the localhost-running app that hosts your data.

If funds become available, Fire Tools may provide an optional server in the future. That server would be completely optional, because personal finance should stay privacy-first by default.
